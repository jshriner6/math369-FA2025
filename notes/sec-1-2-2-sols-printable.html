    2. Augmented Matrices.

    Consider the system
    \begin{equation*} \begin{alignedat}{3} 2x + 4y \amp = \amp 3 \\ -6x - 5y \amp = \amp 1 \\ \end{alignedat} \end{equation*}
    Perform the row operation \(3R_1 + R_2 \to R_2\text{.}\)
    ðŸ”—
    Solution.
    The given row operation results in the system
    \begin{equation*} \begin{alignedat}{3} 2x + 4y \amp = \amp 3 \\ 7y \amp = \amp 10 \\ \end{alignedat} \end{equation*}
    ðŸ”—
    Note that if we changed the variable names this doesnâ€™t fundamentally change our process. The coefficients drive the solution space, but it is important to keep track of their positions with respect to different variables. We can do this by using an augmented matrix.
    ðŸ”—
    As an example, consider the linear system
    \begin{equation*} \begin{alignedat}{3} -x - 2y +2z \amp = \amp -1 \\ 2x+4y - z \amp = \amp 5 \\ x+2y \amp = \amp 3 \\ \end{alignedat} \end{equation*}
    The corresponding augmented matrix would be
    \begin{equation*} \left( \begin{array}{ccc|c} -1 \amp -2 \amp 2 \amp -1 \\ 2 \amp 4 \amp -1 \amp 5 \\ 1 \amp 2 \amp 0 \amp 3 \end{array} \right) \end{equation*}
    The sub-matrix to the left of the vertical line is called the coefficient matrix.
    ðŸ”—
    Row operations on equations extend naturally to row operations on matrices. This is more efficient, but we have to remember where the matrices come from in order to accurately interpret meaning.

    4. Definition: RREF.

    How do we know a matrix is reduced to the "easiest" form?
    ðŸ”—
    We say that a matrix is in reduced row echelon form if the following properties are satisfied.
    • If the entries in a row are all zero, then the same is true of any row below it.
      ðŸ”—
      ðŸ”—
    • If we move across a row from left to right, the first nonzero entry we encounter is 1. We call this entry the leading entry in the row.
      ðŸ”—
      ðŸ”—
    • The leading entry in any row is to the right of the leading entries in all the rows above it.
      ðŸ”—
      ðŸ”—
    • A leading entry is the only nonzero entry in its column.
      ðŸ”—
      ðŸ”—
    We call a matrix in reduced row echelon form a reduced row echelon matrix.
    ðŸ”—
    The following is an example of a matrix in RREF:
    \begin{equation*} \left[ \begin{array}{rrrrrr} 1 \amp * \amp 0 \amp * \amp 0 \amp * \\ 0 \amp 0 \amp 1 \amp * \amp 0 \amp * \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp * \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ \end{array} \right]. \end{equation*}
    ðŸ”—
    Solution.
    The position of a leading entry in a matrix in RREF is called a pivot position.

    8. Observations: Solution Spaces and Pivots.

    ðŸ”—
    Solution.
    A linear system is inconsistent exactly when a pivot position appears in the rightmost column of the augmented matrix.
    ðŸ”—
    A solution is unique when every column of the coefficient matrix contains a pivot position. Columns containing pivot positions correspond to basic variables of the system.
    ðŸ”—
    There are infinitely many solutions when there is a column of the coefficient matrix without a pivot position. Columns not containing pivot positions correspond to free variables of the system.
